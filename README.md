## Setup
```
git clone https://github.com/CoderMike1/weather-app.git
```
```
cd weatherApp
```
Put API Key from [Here!](https://openweathermap.org/) to src/key.json

```
npm install
```
```
npm run build
```
```
npm run dev
```
